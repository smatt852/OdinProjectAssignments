The Odin Project

It is a free, comprehensive curriculum that equips learners to be full stack developers, whatever their experience level. It emphasizes practice and learning by doing. Learning happens while building projects. Topics include HTML, CSS, JavaScript, Accessibility, GitHub and Programming Logic. It is open source and community driven.
